# Portfolio Builders AI Chatbot

A smart AI-powered website assistant for Portfolio Builders. It acts as a friendly career guide, course recommender, and admission assistant.

## Project Structure
- **/backend**: Python FastAPI AI chatbot backend.
- **/frontend-widget/react-widget**: Embedded React Chatbot widget built with Vite.
- **/scripts**: Data scraping (`crawl_website.py`) and vector ingestion (`ingest.py`) scripts.
